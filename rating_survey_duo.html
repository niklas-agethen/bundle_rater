Streamline Bundle Rater: How to use this tool

• Rate the displayed bundles from 0 (not usable) to 4 (highest clinical relevance).
• Provide two rates per screen.
• For AF images: left image = first rating, right image = second rating.
• For CST/OR images: both ratings apply to the same image (rate from visual left to right).
• Current image and rating index are displayed above the rating buttons.
• Use number keys 0–4 for fast rating.
• Skip if no clear rating possible.
• Back lets you revise previous ratings (previous rating will be deleted).
• Next images are loaded automatically.
• Progress is saved automatically.
• Download ratings to download interim results at any time.
• Your final results will be downloaded automatically.
• Start over resets all ratings and starts a new run with random order of images.
