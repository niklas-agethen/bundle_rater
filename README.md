## Streamline Bundle Rater — How to use this tool

- Open html file in a browser to start the rating tool
- **Rate the displayed bundles** from **0 (not usable)** to **4 (highest clinical relevance)**.
- **Provide two ratings per screen**.
- **AF images**:  
  - Left image = **first rating**  
  - Right image = **second rating**
- **CST / OR images**:  
  - Both ratings apply to the **same image**  
  - Rate **from visual left to right**
- The **current image index** and **rating index** are shown above the rating buttons.
- Use number keys **`0–4`** for fast rating.
- Use **Skip** if no clear rating is possible.
- **Back** lets you revise previous ratings  
  *(the previous rating will be deleted)*.
- Next images are loaded **automatically**.
- **Progress is saved automatically**.
- **Download ratings** to export **interim results at any time**.
- Your **final results are downloaded automatically** at the end.
- **Start over** resets all ratings and starts a new run with a **randomized image order**.
